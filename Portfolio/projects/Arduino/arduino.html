Details project

Dit project maakt gebruik van een arduino shield met een 7segment display waarin je een game speelt waarbij je het aangegeven getal moet creeren door op het juiste moment op het knop je te drukken. Het spel bestaat uit meerdere levels en gaat sneller na elke getal. als je 3 fouten heb gemaakt moet je opnieuw beginnen vanaf level0.
